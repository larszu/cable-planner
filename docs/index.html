What's inside

Everything you need to wire a show

One desktop app for the canvas, the routing, the multiview, and the paperwork. No SaaS, no logins, no internet required.

Node-based canvas

React Flow powered drag-and-drop. Pan, zoom, minimap, snap. Visualize the entire signal topology at a glance.

Equipment library

Built-in broadcast gear + custom device templates with proper input/output ports — reusable across every project.

ATEM multiviewer

Visual multiview layout editor for Blackmagic ATEM. Camera / program / preview assignment with exportable configs.

Videohub routing

SDI source → destination patch mapping for Blackmagic Videohub. Logical router editor with visual signal path overview.

Rentman integration

Pull projects, equipment and categories straight from Rentman. Token-based auth, encrypted local storage, zero secrets in code.

PDF export

Print-ready production paperwork. Full cable layouts, equipment lists, metadata and signal routing in one document.

ATEM Multiviewer

Configure the multiview before you patch a single cable

Drag-and-drop layout grid for ATEM Television Studio systems. Tally states, source labels and audio meters all align with what operators see live on monitor walls.

  • 2×2, 4×4 and asymmetric program/preview layouts
  • Per-cell source assignment with live preview labels
  • Hardware tally indication (PGM red, PVW green)
  • Export-ready ATEM configurations

Videohub Routing

Patch sources to destinations visually

The same matrix UI that lives inside CablePlanner — click any cell to route a source to a destination. Active routes are highlighted, mismatches are flagged.

  • Logical router configuration for any Blackmagic Videohub size
  • Click-to-route patch matrix with hover preview
  • Save and recall full snapshots per show
  • Exportable routing setup files
