# Suite-Mockup — Design-Inspiration fuer die gemeinsame App-Shell

Frühe HTML-Design-Studie ("Broadcast Suite") einer vereinheitlichten Shell, die
Cable-, Multicam- und Light-Planner unter einer Oberfläche zusammenführt.

**Nur Inspiration, kein Zielbild** — die echte Shell soll die guten Ideen
aufgreifen und per Recherche (online + im Repo) darüber hinausgehen, NICHT 1:1
nachgebaut werden.

## Inhalt
- `suite.html` — das statische Mockup (Fake-Daten)
- `suite-signal.png` — Signal-Flow-Ansicht (Kabel/Route/BOM)
- `suite-kameras.png` — Kamera-Modul
- `suite-licht.png` — Licht-Modul

## Inspirations-Bausteine
Modul-Leiste links (Raum/Kameras/Licht/Signal/Ebenen/Bibliothek), gemeinsame
Topbar mit Command-Palette (⌘K), zentrales Tab-Deck (Signal-Flow/2D-Plan/Rack),
kontextsensitives Eigenschaften-Panel rechts (Plan-Check-/BOM-/Rentman-Badges),
untere Statusleiste, Venue-Geräte automatisch verknüpft, Dark-Theme mit Akzent.
